# Documentación Técnica Global - Coverly

## Propósito

Este directorio contiene la documentación técnica de alto nivel del sistema Coverly.
Está diseñada para proporcionar contexto global a:

- Desarrolladores del equipo
- Nuevos integrantes
- Agentes de inteligencia artificial utilizados en desarrollo
- Procesos de integración futura
